#Firmware
This hackpad uses [QMK](https://qmk.fm/) firmware for everything. 
-The rotary encoder cahnges volume. 
-The keys act as shortcuts for opening apps, exectuing commands, ...
-The OLED shows the time.


#BOM:
Here you can find all the parts you need for this hackpad

- 4x Cherry MX Switches
- 4x DSA Keycaps
- 8x WS2812B LEDs
- 1x 0.91" 128x32 OLED Display
- 1x EC11 Rotary Encoder
- 1x XIAO RP2040
- 1x Case (3 printed parts)
